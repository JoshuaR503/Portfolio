In 2019, I worked for a startup called CatalogIt. I developed 4 WordPress plugins which consumed CatalogIt's API and displayed content to the user's WordPress website.

The plugins allow the user's to display content in two different modes, a grid or a list. The plugins also allow the user to search entries in folders on the whole user's account. The plugins support EI11 which became a challenge, especially when some of the needs were to have infinite scrolling and lazy loading of pictures. If you would like to see a live example, please visit American Bookbinders Museu online Catalog.
