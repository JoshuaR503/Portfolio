Vocabulary Builder Dashboard was originally built with Angular.js but soon after the lack of experience and clumsy code that I wrote, I decided to change to Vue.js. Vue.js was a simple and easy-to-use framework that fitted the needs of the Dashboard panel.

The Dashboard became a need after building the Vocabulary Builder Server which needed a graphic interface. Before building the Dashboard I would use Postman to make CRUD operations. The Dashboard serves to monitor bugs, add and update content, and read user's feedback left from the app.
