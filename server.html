Vocabulary Builder Server is a Node.js based server which is was built in TypeScript and the Express.js framework. The server's code went under a massive restructuring to improve its reliability and maintance. It was originally built with JavaScript, but after a series of bugs and securities concerns, I decided to switch everything into the static-typed programming language TypeScript. The server provides an API to feed content into The Vocabulary Builder app.
